# 1.快捷键

| 快捷键                   | 说明             |
| ------------------------ | ---------------- |
| F1或者Ctrl + shift + p   | 打开命令面板     |
| Alt + ←                  | 回到上一个位置   |
| Alt + shift + f          | 格式化代码       |
| Ctrl + shift + f         | 搜索选中内容     |
| Ctrl + \                 | 分割编辑窗口     |
| Alt + shift + up或者down | 复制一行         |
| Alt + up或者down         | 上下移动一行     |
| Alt + shift + →或者←     | 扩大或者缩小选择 |
